# The `1-my-list` Module
=========================

Using `print_sorted()` function from the `1-my_list` module.
_________________________________________________________

>>> MyList = __import__('1-my_list').MyList

>>> my_list = MyList([1, 4, 2, 3, 5])

>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

>>> print(my_list)
[1, 4, 2, 3, 5]

>>> my_list.append(-100)

>>> my_list.print_sorted()
[-100, 1, 2, 3, 4, 5]

>>> print(my_list)
[1, 4, 2, 3, 5, -100]

>>> copy = my_list.print_sorted()
[-100, 1, 2, 3, 4, 5]

>>> copy is my_list.print_sorted()
[-100, 1, 2, 3, 4, 5]
True

>>> my_list = MyList()

>>> my_list.print_sorted()
[]

>>> print(my_list)
[]
